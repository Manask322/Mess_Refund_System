User Documentation

Introduction

This web application enables the user (student) to keep track of the amount spent for mess food and hence, the refund they need to receive by the end of the semester, along with daily menu of the chosen mess. It offers auto-scaling and high availability, supports all platforms as it works on browser. Learn how to use the application with this documentation.

Getting Started

Profile Details: Editing your details as you see fit. It re-directs you to the home page after updating the details. You can always come back to this page navigating from the sidebar, which can be seen at the top left side of the navigation bar.

The home page provides you with the basic details of the application along with the respective contacts in case of any issues. We assure you that the response would be immediate.

Side Bar: There are four functions that this navigation bar does for you. - Navigate to edit your profile details through 'Profile Details' link. - Scan the QR code
